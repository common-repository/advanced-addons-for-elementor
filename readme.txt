== Installation ==

It's really easy to install Advanced Addons for Elementor plugin, but before installing it make sure you've installed [Elementor](https://wordpress.org/plugins/elementor/ "Install Elementor"). Elementor is the only dependency.

= Minimum Requirements =

* WordPress 4.7 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.0 or greater
* MySQL version 5.6 or greater
* WordPress Memory limit of 128 MB or greater


= Automatic Installation =
1.  Go to `Plugins > Add New` screen in WordPress.
2.  Search for `Advanced Addons for Elementor`.
3.  Install and activate the plugin, that's it.

= Manual Installation =
1.  Download [Advanced Addons for Elementor](https://downloads.wordpress.org/plugin/advanced-addons-for-elementor.zip "Download Advanced Addons for Elementor").
2.  Extract the `'advanced-addons-for-elementor'.zip` file. You'll get plugin files inside `'advanced-addons-for-elementor'` directory.
3.  Upload the plugin files to the `/wp-content/plugins/'advanced-addons-for-elementor'` directory.
4.  Activate the plugin through the 'Plugins' screen in WordPress.
